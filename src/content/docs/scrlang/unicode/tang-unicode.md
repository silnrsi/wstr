In The Unicode Standard, Tangut script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch18.pdf Chapter 18 East Asia}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

The Tangut script was encoded in The Unicode Standard version 9.0. The script is now encoded in the following blocks:

table:
Blocks|Character Range|Added in Unicode Version|Unicode chart|
Ideographic Symbols and Punctuation |16FE0..16FFF|9.0|{link:http://www.unicode.org/charts/PDF/U16FE0.pdf U16FE0}|
Tangut |17000..187FF|9.0|{link:http://www.unicode.org/charts/PDF/U17000.pdf U17000}|
Tangut Components|18800..18AFF|9.0|{link:http://www.unicode.org/charts/PDF/U18800.pdf U18800}|
Tangut Supplements|18D00..18D7F|13.0|{link:http://www.unicode.org/charts/PDF/U18D00.pdf U18D00}|
