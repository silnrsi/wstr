This table displays the transliteration systems used by [the American Library Association - Library of Congress (ALA-LC)][ala-lc-ethi], Alone-Stokes, Bright and Daniels, Campbell, Chaîne, Cohen, Dawkins, US Department of State, and [the ethiop TeX package][ethiop]. It also shows the transcription system used by [The System for Ethiopic Representation in ASCII (SERA)][sera].

The table is ostensibly for the Amharic language, however it seems clear that at least a few of the resources were extended to handle Ethiopic characters in use by other Ethiopian languages. For example, Chaîne mentions characters in the Tigrai language, and ethiop and SERA have extensive additions beyond Amharic.
