If a font supports all of the following characters, we believe that all IPA requirements, most phonetic, linguistic, transliteration and transcription uses (in Latin script) would be met. Note, however, that this list does _not_ include required characters for orthographic description of languages, just phonetic, linguistic or transliteration use.

There are comments below about glyph variants that may be needed to support some instances. 

The process for developing this list was based on the following (in order of process):

- A review of custom-encoded phonetic fonts
- A review of the IPA handbook including both official and unofficial characters
- A review of the Library of Congress transliteration tables
- A review of annotations in [The Unicode Standard charts][uni-charts]
- Decision to include all Codepage 1252 and MacRoman characters
- Decision to include many characters required for publishing (spaces and dashes)
- Decision that if a combining mark combining mark, such as :usv[0302]{usv char name} was required, then all precomposed characters with that mark should also be included. (e.g., if :usv[0061]{usv char name} was in the subset, then :usv[00E2]{usv char name} was also included in the subset, but since neither :usv[0309]{usv char name} nor :usv[1EA3]{usv char name} were in the subset, :usv[1EA9]{usv char name} was not included in the subset).
- Feedback on the resulting character set
