The Box-Headed script is not yet in Unicode. The script has a tentative allocation at U+1E660..U+1E6BF in the [Roadmap to the SMP](http://www.unicode.org/roadmaps/smp/) for the Unicode Standard.

## Blocks

None.

## Characters

None.
