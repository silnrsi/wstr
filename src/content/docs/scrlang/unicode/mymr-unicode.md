In The Unicode Standard, Myanmar script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch16.pdf Chapter 16 Southeast Asia} and in {link:http://www.unicode.org/notes/tn11/ Unicode Technical Note #11: Representing Myanmar in Unicode}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

The Myanmar script was encoded in The Unicode Standard version 3.0. Since that time the encoding has undergone a number of modifications; the script is now encoded in the following blocks:

table:
Blocks|Character Range|Added in Unicode Version|Unicode Chart|
Myanmar|1000..109F|3.0|{link:http://www.unicode.org/charts/PDF/U1000.pdf U1000.pdf}|
Myanmar Extended-B|A9E0..A9FF|7.0|{link:http://www.unicode.org/charts/PDF/UA9E0.pdf UA9E0.pdf}|
Myanmar Extended-A|AA60..AA7F|5.2|{link:http://www.unicode.org/charts/PDF/UAA60.pdf UAA60.pdf}|
