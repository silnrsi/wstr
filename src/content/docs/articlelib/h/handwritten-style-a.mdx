When people are looking for a handwritten style of the letter "a" they might be tempted to use :usv[0251]{usv char} instead of :usv[0061]{usv char} because it looks more like handwriting. :usv[0251]{usv char name} is in Unicode as an IPA symbol. It should not be used instead of an "a". A font should be used which has a handwritten style "a" at :usv[0061]{usv char name}. If you use :usv[0251]{usv char name} you will have unexpected results with data analysis as well as when using uppercase/lowercase pairs.
