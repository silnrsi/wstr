## Script description

The Cypro-Minoan script was a logosyllabary used around 1500-1200 BC for writing the Eteocretan language.

<details>
<summary>Read the full description...</summary>The name of the script reflects the possibilities that it derived from a Minoan script and is ancestral to the [Cypriot](/scrlang/scripts/cprt) script.
</details>

## Unicode status

The Cypro-Minoan script is not yet in Unicode. The script has a tentative allocation at U+12700..U+1278F in the [Roadmap to the SMP](http://www.unicode.org/roadmaps/smp/) for the Unicode Standard.
