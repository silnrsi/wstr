Kodagu scholar Dr Iychettira M Muthanna invented an alphabet for the Kodava language in 1970. Appaneravanda Kiran Subbaiah invented a different script for the Kodava language in 1980. 

In February 2022, Karnataka Kodava Sahitya Academy, the local government cultural and lingual body, recognised the Muthanna script as the official script because it was easiest to learn.
