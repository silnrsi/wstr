import ApiBrowser from '../../../components/ApiBrowser'

Type in a [BCP47 language tag (langtag)](/topics/writingsystems/language-tagging)
e.g. *nod-Lana-TH, wsg-Gong-IN, shu-Arab-TD, taq-Tfng-ML* (using the *tag-Script-REGION* format)
to find corresponding font information.

## Technical details
See the [LFF service API documentation](https://lff.api.languagetechnology.org/docs) for more details on API access.

To report issues with the data returned please create an [issue](https://github.com/silnrsi/langfontfinder/issues) with the Language Font Finder project GitHub repository.

To report issues with this page please create an [issue](https://github.com/silnrsi/wstr/issues) with thw WSTR project GitHub repository
