In The Unicode Standard, Brahmi script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch14.pdf Chapter 14 South and Central Asia-III: Ancient Scripts}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

The Brahmi script was encoded in The Unicode Standard version 6.0. The script is encoded in the following block:

table:
Blocks|Character Range|Added in Unicode Version|Unicode Chart|
Brahmi|11000..1107F|6.0|{link:http://www.unicode.org/charts/PDF/U11000.pdf U11000}|

[comment]: # (end of blocks)

[comment]: # (start of chars)

Subsequent to version 6.0, the following characters have been added to the Brahmi script:

table:Documentation refers to {link:https://www.unicode.org/wg2/ ISO Working Group} and Unicode proposals
Characters|Unicode Version|Documentation|
1107F|7.0|{link:https://www.unicode.org/wg2/docs/n4166.pdf WG2 N4166}, {link:http://www.unicode.org/cgi-bin/GetMatchingDocs.pl?L2/11-357 L2/11-357}|
