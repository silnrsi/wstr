## Script description

Todhri script is attributed to Theodor Haxhifilipi of Elbasan in central Albania, and was used to write the Albanian language.

Todhri consists of 52 characters.

<details>
<summary>Read the full description...</summary>Todhri was used in the 18th  and the 19th centuries to write Biblical texts. Merchants also used the script for correspondence and record-keeping.
</details>
