In The Unicode Standard, Sharada script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch15.pdf Chapter 15 South and Central Asia-IV}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

The Sharada script was encoded in The Unicode Standard version 6.1. The script is encoded in the following block:

table:
Blocks|Character Range|Added in Unicode Version|Unicode Chart|
Sharada|11180..111DF|6.1|{link:http://www.unicode.org/charts/PDF/U11180.pdf U11180}|

{link:https://scriptsource.org/entry/nb2xvfymgv Vedic Extensions} may be used with the Sharada script as well as many other Indic scripts.

[comment]: # (end of blocks)
