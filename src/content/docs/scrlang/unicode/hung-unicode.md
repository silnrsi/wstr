In The Unicode Standard, Old Hungarian script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch08.pdf Chapter 8 Europe-II}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

The Old Hungarian script was encoded in The Unicode Standard version 8.0. The script is encoded in the following block:

table:
Blocks|Character Range|Added in Unicode Version|Unicode Chart|
Old Hungarian|10C80..10CFF|8.0|{link:http://www.unicode.org/charts/PDF/U10C80.pdf U10c80}|
