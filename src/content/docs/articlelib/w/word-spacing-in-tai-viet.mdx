Traditionally, the Tai Viet script was written without spaces between words. In the last 30 years, users in both Vietnam and the United States have started writing spaces between words, in both hand written and machine produced texts. Most users now use interword spacing. However, considering the script’s historical usage, it was felt that a set of line breaking rules should be provided for instances where interword spacing is not desired.
