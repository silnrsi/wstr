Romanian used to use a glyph variant with a comma below (rather than cedilla) for the following characters: :usv[015E]{usv char name}, :usv[015F]{usv char name}, :usv[0162]{usv char name}, and :usv[0163]{usv char name}.

![old-glyph-variants-for-romanian-1](images/o/old-glyph-variants-for-romanian-1)

It is now recommended that Romanian data use :usv[0218]{usv char name}, :usv[0219]{usv char name}, :usv[021A]{usv char name}, and :usv[021B]{usv char name} which do not require glyph variants.
