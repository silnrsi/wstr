## Script description

The Cypro-Minoan script was a logosyllabary used around 1500-1200 BC for writing the Eteocretan language.

<details>
<summary>Read the full description...'</summary>The name of the script reflects the possibilities that it derived from a Minoan script and is ancestral to the [Cypriot](/scrlang/script-cprt) script.
</details>
