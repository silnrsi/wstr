The Oduduwa (sometimes called Yoruba Ifa or Aladura Holy Alphabet) script is not yet in Unicode. The script has not yet been added to the [Roadmap to the SMP](http://www.unicode.org/roadmaps/smp/) for the Unicode Standard. 

## Blocks

None.
