The Mwangwego has been accepted for encoding in a future version of the Unicode Standard. The script has a provisional allocation at U+16E00..U+16E3F in the [Roadmap to the SMP](http://www.unicode.org/roadmaps/smp/) for the Unicode Standard.

## Blocks

None.
