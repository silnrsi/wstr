Partial transcription:

2. .... niun 9 lai<br />
3. sip song wan wan tai kap mut<br />
4. wan tham so kra po hin

2. ‘... in the ninth month, getting<br />
3. ‘the 12th day, the Tai day Kap Mut<br />
4. ‘the Dhamma day Friday ...’

Whilst this is written in Tai Le script, it is very similar to Ahom script, indicating a close relationship between the two.
