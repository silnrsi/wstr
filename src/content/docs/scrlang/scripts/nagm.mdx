## Script description

The Nag Mundari script has sometimes been called _Mundari Bani script_.

<details>
<summary>Read the full description...</summary>It has been used for writing the Mundari language of India. The script was invented by Rohidas Singh Nag in the latter half of the 20th century.

The script has 27 letters, 5 "tong" (a collection of symbols and diacritics), and script specific digits. European-style punctuation is used.
</details>
