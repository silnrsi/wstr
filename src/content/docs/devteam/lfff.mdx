---
title: Language Font Finder
description: A tool to lookup font information for a given language.
---
import ApiBrowser from '../../../components/ApiBrowser'

Type in a [BCP47 language tag (langtag)](/topics/writingsystems/language-tagging)
e.g. *nod-Lana-TH, wsg-Gong-IN, shu-Arab-TD, taq-Tfng-ML* (using the *tag-Script-REGION* format)
to find corresponding font information.

See the [LFF service API documentation](https://lff.api.languagetechnology.org/docs) for more details.
