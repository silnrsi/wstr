If a font supports all of the following characters, we believe that most languages of Asia and the Pacific region (that use the Latin script) would be able to typeset texts in their orthography. Note, however, that this list does _not_ include required characters for phonetic description of all Asia/Pacific languages, just orthographic use.

There are comments below about glyph variants that may be needed to support some languages. 

The process for developing this list was based on the following (in order of process):

- A review of custom-encoded fonts from Asia/Pacific
- Decision to exclude Vietnamese (see [separate character subset for Vietnamese](/articlelib/c/character-requirements-for-a-vietnamese-font/))
- 20+ years' experience of user support for language support from Asia/Pacific
- A review of annotations in [The Unicode Standard charts][uni-charts]
- Decision to include all Codepage 1252 and MacRoman characters
- Decision to include of many characters required for publishing (spaces and dashes)
- Decision that if a combining mark combining mark, such as :usv[0323]{usv char name} was required, then all precomposed characters with that mark should also be included. (e.g., if :usv[0061]{usv char name} was in the subset, then :usv[1EA1]{usv char name} was also included in the subset, but since :usv[01A1]{usv char name} was _not_ in the subset, then :usv[1EE3]{usv char name} was not included in the subset).
- Feedback on the resulting character set
