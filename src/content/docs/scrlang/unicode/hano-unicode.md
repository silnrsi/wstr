In The Unicode Standard, Hanunóo script implementation is discussed in [Chapter 17: Southeast Asia-II — Indonesia and the Philippines](https://www.unicode.org/versions/latest/core-spec/#G26437).

## Blocks

The Hanunóo script was encoded in The Unicode Standard version 3.2. The script is encoded in the following block:

| Blocks | Character Range | Added in Unicode Version | Unicode Chart |
| ------ | --------------- | ------------------------ | ------------- |
| Hanunóo | 1720..173F | 3.2 | [U1720.pdf](http://www.unicode.org/charts/PDF/U1720.pdf) |
