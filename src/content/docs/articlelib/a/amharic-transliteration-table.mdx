This table displays the transliteration systems used by [the American Library Association - Library of Congress (ALA-LC)](http://www.loc.gov/catdir/cpso/romanization/amharic.pdf), [Alone-Stokes](https://scriptsource.org/source/uyq95xnldz), [Brights and Daniel](https://scriptsource.org/source/3cg3de97db), [Campbell](https://scriptsource.org/source/ja5hfkr9na), [Chaîne](https://scriptsource.org/source/mjtyl2btt4), [Cohen](https://scriptsource.org/source/te3bw27bzl), [Dawkins](https://scriptsource.org/source/hza9ttlulg), [US Department of State](https://scriptsource.org/source/yzf55ffv3h) and [the ethiop TeX package](https://scriptsource.org/source/ksj88tpn9w).  It also shows the transcription system used by [The System for Ethiopic Representation in ASCII (SERA)](https://scriptsource.org/source/hempk98rlm).

The table is ostensibly for the Amharic language, however it seems clear that at least a few of the resources were extended to handle Ethiopic characters in use by other Ethiopian languages. For example, Chaîne mentions characters in the Tigrai language, and ethiop and SERA have extensive additions beyond Amharic.
