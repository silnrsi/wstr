There are two lowercase schwa characters and you might think either one will work for you. They are :usv[0259]{char usv} and :usv[01DD]{char usv}.

This one will rise up and bite you if you are not careful! This is where looking at the Unicode documentation is important. If you look at the [chart](http://www.unicode.org/charts/PDF/U0250.pdf) for :usv[0259]{usv} you will see:

![choosing-the-right-schwa-1](images/c/choosing-the-right-schwa-1)

There are a number of useful bits of information here. The second bullet tells you "uppercase is :usv[018F]{usv char}". This tells us that :usv[018F]{usv} is the uppercase match to this character. Another bit of useful information is right below that: "variant uppercase form :usv[018E]{usv char} is associated with :usv[01DD]{usv char}."

Another interesting test is to type both of the schwas into a word processor (like Word). Select them both and click on <span class='menu_choice'>Format / Change Case... / UPPERCASE</span>. You should see two different forms of the uppercase schwa. This shows you how important it is to match the lowercase character (which _looks_ exactly the same) with the correct uppercase character (which looks significantly different).

![choosing-the-right-schwa-2](images/c/choosing-the-right-schwa-2)

In this example you want to make sure that if you are using :usv[018F]{usv char} in your orthography, you should use :usv[0259]{usv char} for the lowercase.
