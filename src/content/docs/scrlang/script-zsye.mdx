## Script description

The classification “Symbols” is used for characters which do not belong to a language, and which do not fit another designation in the ISO 15924 standard.

<details>
<summary>Read the full description...</summary>Recently, “Emoji variant” has been added as a new classification of “Symbols”. Emoji are “picture characters” originally associated with mobile phone use in Japan, but they are now popular worldwide.
</details>
