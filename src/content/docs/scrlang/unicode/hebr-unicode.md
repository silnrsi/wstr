In The Unicode Standard, Hebrew Script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch09.pdf Chapter 9 Middle East-I: Modern and Liturgical Scripts}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

Hebrew script was first encoded in The Unicode Standard version 1.0. The script is encoded in the following blocks:

table:
Blocks | Character Range | Added in Unicode Version | Unicode Chart |
Hebrew | 0590..05FF | 1.0 | {link:http://www.unicode.org/charts/PDF/U0590.pdf U0590.pdf} |
Alphabetic Presentation Forms | FB10..FB4F| 1.1| {link:http://www.unicode.org/charts/PDF/UFB00.pdf UFB00.pdf|
