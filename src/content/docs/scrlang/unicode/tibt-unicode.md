In The Unicode Standard, Tibetan script implementation is discussed in {link:http://www.unicode.org/versions/latest/ch13.pdf Chapter 13 South and Central Asia-II: Other Modern Scripts}.

[comment]: # (end of intro)

[comment]: # (start of blocks)

The Tibetan script was first encoded in The Unicode Standard version 2.0. The script is encoded in the following block:

table:
Blocks|Character Range|Added in Unicode Version|Unicode Chart|
Tibetan|0F00..0FFF|2.0|{link:http://www.unicode.org/charts/PDF/U0F00.pdf U0F00.pdf}|

[comment]: # (end of blocks)
