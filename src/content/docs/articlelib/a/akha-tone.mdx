Akha is a language spoken by close to half a million people in Myanmar, China, Laos, Thailand and Viet Nam. Akha has oral vowels occurring on three level tones:

|  |  |
| --- | --- |
| / ˇ / | high | U+02C7 CARON |
| /   / | mid (not marked) |  |
| / ˬ / | low | U+02EC MODIFIER LETTER VOICING |

and largyngealized vowels occur on two level tones:

|  |  |
| --- | --- |
| / ˆ / | mid | U+02C6 MODIFIER LETTER CIRCUMFLEX ACCENT |
| / ꞈ / | low | U+A788 LETTER LOW CIRCUMFLEX ACCENT |
