In The Unicode Standard, Emoji implementation is discussed in {link:http://www.unicode.org/versions/latest/ch22.pdf Chapter 22 Symbols}.

{link:http://unicode.org/reports/tr51/ Unicode Technical Report #51} discusses Unicode Emoji, and there are strict guidelines on which characters might be accepted in the future: {link:http://unicode.org/emoji/selection.html}.
