:::caution[Test version]{icon="sun"}
This is an early beta test version. We welcome your feedback and suggestions - see the Feedback section below. Please let us know if this service is helpful to you, and if there are any features you would like to see added.
:::

Type a language name or [language tag][language-tagging] in the box below to choose a language. Then select the script, and the _Language Font Finder_ will provide information about free and openly licensed fonts that are likely to support that writing system.

_Known issue with this beta version: Multi-word searching is not working properly. If you want to search for a language with multiple words in the name, try using the [language tag][language-tagging] instead of the name, or try searching for just the first word in the name._

<LFFBrowser client:only="react"/>

The data used for this _Language Font Finder (LFF)_ is also available to software developers through a REST API. See the [LFF service API documentation](https://lff.api.languagetechnology.org/docs) for details. Some sample texts are from the [Google Fonts language data][googlefontstexts] and used under the terms of the [Apache 2.0 license][apache-2.0].

## Feedback

We would love your feedback on this new tool. Please let us know if you find it useful.

If you encounter any problems with this tool, or have suggestions for improvements, please create an issue in the [WSTR project GitHub repository][wstr-issues].

To report issues with the font or language data, please create an issue in the [Language Font Finder project GitHub repository][lff-issues].

If you would prefer a different way to contact us, please see the [Contact page][contact] for other options.
