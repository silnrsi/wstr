Tai Viet characters are not sorted in any particular order. Neighbouring languages have sometimes influenced a decision to adopt a particular order. Some dictionaries for example have used a modified Laotian order, others use a modified Vietnamese order. The order represented in the Unicode chart is based on the Lao order. It is possible that when the matter is discussed at length with the Tai people, the conclusion will be that communities in different countries prefer different orders.
