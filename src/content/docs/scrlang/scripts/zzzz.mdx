## Script description

Scripts on this page are not currently recognized by [ISO 15924](http://www.unicode.org/iso15924/), _and_ we do not currently have individual script pages for many of them.

<details>
<summary>Read the full description...</summary>We also include scripts which have been rejected for encoding. If you have any information on these scripts, please add the information to the site. Your contributions can be a great help in refining and expanding the ISO 15924 standard.
</details>

## Languages that use this script

:::note
 The name of the script is indicated by the regional variant name (e.g., Adinkra, Badaga, Stokoe, etc.). A status of _obsolete_ indicates that the writing system is no longer in use for that language; the language may still be spoken.
:::
